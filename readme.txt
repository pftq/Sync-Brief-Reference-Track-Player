Sync Brief Reference Track Player (Chrome extension for gmail)
by pftq

Handy tool for playing the reference track automatically via YouTube on sync brief emails by The Sync Opps.

Install instructions:
1. Go to chrome://extensions  
2. Enable Developer mode
3. Click Load unpacked
4. Select this folder

This tool is originally targetted for emails from The Sync Opps but can be customized to any format if you adjust the REGEX patterns in the patterns.txt file. You can also select text and right-click to play the YouTube result for that text directly.
